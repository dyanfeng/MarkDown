# demo01

## 标题
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
```
## 段落
```
注意:
1. 段落和段落之间要有空行,否则会连续在一行显示
2. 如果觉得两行之间的间距太大,可以在第一行写完后,敲两个空格再回车
```

## 强调
```
*我是斜体*
**我是粗体**
***我是粗体加上斜体***
~~我是波浪线~~
```

## 列表

### 无序列表
```
* 第一行
* 第二行
* 第三行

或者等价于

- 第一行
- 第二行
- 第三行

注意:无序列表可以嵌套,嵌套时,下一行前加上两个空格
```
### 有序列表
```
1. 第一行
  1. 第一小行
  2. 第二小行
2. 第二行
3. 第三行
```
